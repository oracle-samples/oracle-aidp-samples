# Agent Flows Samples

We provided a collection of agent flows spanning different use cases and industries. We included step-by-step instructions for agent flows built in the visual flow experience (`visual-flow/`) as well as agent flows built through the code-first (`code-first/`) experience.

For the code samples, beware that you will need to provide your own OCI tenancy relevant information in the Python files. These parameters are highlighted throughout the code. Additional instructions to set the entry file and the dependency files are also provided whenever necessary.

## Sub-folders

| Folder | What's inside |
|---|---|
| [visual-flow/](./visual-flow/) | Walkthroughs for flows built in the canvas UI (hello-world, acme-insurance, entertainment-analyst, utility-operator). |
| [code-first/](./code-first/) | Flows authored in Python with the code-first SDK. |
| [custom-tools/](./custom-tools/) | Python custom tool packages that plug into any agent flow — hello tool, a developer toolkit (bash/file/python), and an ORDS database tool. Start with [`custom-tools/USER_GUIDE.md`](./custom-tools/USER_GUIDE.md). |
| [misc/](./misc/) | Related notebooks (e.g. scheduled triggers for agent flows). |
