# Simple ReAct Agent with RAG Tool 

## Instructions

1) Upload the PDF files from the data/ folder into an AIDP Workbench standard catalog volume. 
2) Create a knowledge base with the files you just uploaded to a volume. Add the volume as a data source. 
3) Create a new agent flow using the code authoring mode. 
4) Upload the content of this folder to your agent flow folder.
5) In the Python file react-with-rag-tool.py, replace the values of your tenancy OCID, region, catalog, schema, and knowledge base names. 
6) Set the file react-with-rag-tool.py as your entry file. 
7) Attach an AI compute. 
8) Run the Python file. 
9) Test the agent by clicking on the test tab of your agent flow.
